### Admin recalc endpoint (2026-04-17)
Forced refresh without waiting for Kafka events:
```
POST /data/v1/admin/metrics/recalculate?metric_type=all&period=all
Header: X-Admin-Token: <INTERNAL_API_TOKEN>
```
Returns structured summary `{snapshots_written, recalculated{type:[periods]}, scanned, errors}`.
Shared service `src/contexts/metrics/services/recalculate.py` is the single source of truth for both the Kafka-driven worker and this admin path.
